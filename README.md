## 요구사항
Python 3.11.8
파이썬 가상환경
````
# 가상환경 생성
python -m venv venv

# 가상환경 활성화 (Windows)
venv\Scripts\activate

# 가상환경 활성화 (macOS/Linux)
source venv/bin/activate

pip install -r requirements.txt

API 서버 실행 
python main.py
```
